Example: Video with Subtitles

To watch this video, you need the latest Flash-Player and active javascript in your browser.

Available adjustments for the subtitles:

  • subtitlesURL: the url of the subtitles file (you can use .srt or .xml files, see the examples for formatting)
  • subtitlesPosition: "top" or "bottom"
  • subtitlesColor: the color of the subtitles
  • subtitlesSize: guess what, it's the size of the subtitles ;)

See the souce-code of this site if you want to know how the movie was embedded.
